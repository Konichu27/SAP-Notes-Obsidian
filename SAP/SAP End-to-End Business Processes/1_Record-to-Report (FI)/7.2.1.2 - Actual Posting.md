- Refers to cost activities beyond overhead (secondary)
## Direct Activity Allocation
![[Pasted image 20260220211335.png]]
- Involves the **sender cost center**, **receiver cost object**, **activity type**, and **activity quantity**
	- Sender cost center - *credited*
	- Receiver cost object - *debited*
	- The above two are done in a *secondary G/L account*.
	- Activity Allocation Value = Activity Quantity * Planned Price

## WBS Elements
![[Pasted image 20260220211439.png]]
- All projects in SAP are managed using **WBS Elements**.
- **WBS Elements** - act as a "bucket" for costs & revenues
	- Each element of the project can be subdivided into more elements.
	- Each element can have different settings/properties, ex. presence of costs, revenues, and/or budget
