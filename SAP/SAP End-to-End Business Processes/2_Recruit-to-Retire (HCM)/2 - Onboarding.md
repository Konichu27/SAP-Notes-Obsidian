## Onboarding steps

| **New Hire**                                                                                                                                        | *both*                                                       | **Hiring Manager**                                                                                                                        |
| --------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
|                                                                                                                                                     |                                                              | 1. Hiring manager receives **email** to start the onboarding process.<br><br>Additional tasks can also be assigned to the new hire        |
| 2. The new hire receives the **login credentials and URL** for SAP SuccessFactors.<br><br>Tasks assigned can be completed under "**For You Today**" |                                                              |                                                                                                                                           |
|                                                                                                                                                     | 3. New hire and hiring manager **review and sign documents** |                                                                                                                                           |
|                                                                                                                                                     |                                                              | 4. Hiring manager must list additional employee data in **Employee Central**.<br><br>The new hire should appear in *Manage Pending Hires* |
| 5. Status and employee information **are activated**                                                                                                |                                                              |                                                                                                                                           |
