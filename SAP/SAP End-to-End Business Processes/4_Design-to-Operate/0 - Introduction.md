
- Done in **SAP Integrated Business Planning** (IBP)
## Production Process Steps
1. **Master Data / Product Cost Calculation**
2. **Demand Planning**
3. **Procure to Receipt**
4. **Production Order to Fulfill**
5. **Reporting**
## SAP IBP
- Three phases:
	- **Reporting**
	- **Planning**
	- **Executing**
- Has integration with Microsoft Excel/web-based interfaces

### Components of SAP IBP

| **Component**                      | **Description**                                             |
| ---------------------------------- | ----------------------------------------------------------- |
| **SAP Supply Chain Control Tower** | Covers end-to-end visibility, KPIs, alerts, cases & tasks   |
| **Sales and Operations Planning**  | Sales decision processes                                    |
| **Demand**                         | Forecast & demand planning                                  |
| **Inventory**                      | "Multi-stage Inventory Optimization"; inventory plans, etc. |
| **Response and Supply**            | Response plan, scenario planning, planning runs             |
| **Demand-Driven Replenishment**    | Material Requirements Planning (MRP); the core of SAP IBP   |
