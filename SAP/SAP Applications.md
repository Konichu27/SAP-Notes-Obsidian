- **SAP Solution Manager**, **SAP Cloud ALM** (Application Lifecycle Management)
	- For implementing SAP Cloud solutions
	- Help to accelerate time-to-value and reduce total cost of ownership (TCO)
- **SAP Signavio** - suite of applications for business process transformation (BPT)
	- **Process Manager** - process modeling. Like Microsoft Visio but more sophisticated
	- **SAP Signavio Process Navigator** - for business process design
		- **Solution scenario** - documentation for each solution
			- **Accelerators** - resources used to support each process
		- **Solution process** - start-to-finish steps involved in an actual process
			- Also known as a *business process*
		- **Solution process flow diagram**
		- **Test script** - steps to complete an actual process
		- **Task script** - step-by-step simulations for the end-user handling a business process in SAP Cloud ERP
	- **Process Governance** - converts processes made in Process Manager into actual governance overflows
	- **Process Explorer** - preconfigured best-practice processes!
		- Ex. Enterprise Management Layer
	- **Process Collaboration** - for continuous commenting
	- **Journey Modeler** - identifies areas for improvement and automation
	- **Process Insights**
	- **Process Intelligence**
- **SAP Activate** - methodology for SAP S/4HANA implementation
- **WalkMe** - digital adoption platform for end-users using the application
	- Walkthroughs/tutorials
	- Identifies workflow issues & automation
	- Records and analyzes user behavior

		- On-screen recommendations for buying
